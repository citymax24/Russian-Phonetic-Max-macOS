Installation erfolgreich!

Russian Phonetic Max wurde installiert. Bitte führe die folgenden Schritte aus, um das Layout zu aktivieren:

Schritt 1: Alte Layouts entfernen
Systemeinstellungen → Tastatur → Eingabequellen → alle vorhandenen Russisch-Layouts entfernen (z.B. „Russian - Phonetic“).
Wichtig: Mac neu starten (nicht nur abmelden – ein vollständiger Neustart ist erforderlich!).
Schritt 2: Layout hinzufügen
Nach dem Neustart: Systemeinstellungen → Tastatur → Eingabequellen → + → Russisch → „Russian - Phonetic - Max“ auswählen.
Schritt 3: Testen
Layout umschalten und prüfen:
Y → ы    Z → з    ISO-Taste (links neben Shift) → ё
