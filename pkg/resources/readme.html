Was wird installiert?

Datei Ziel
Russian-Phonetic-Max.bundle /Library/Keyboard Layouts/
Russian-Phonetic-Max.keylayout /Library/Keyboard Layouts/

Zusätzlich wird eine Kopie nach ~/Library/Keyboard Layouts/ gelegt (für maximale Kompatibilität mit macOS 26).

Was passiert automatisch?

  • Alte Versionen (Russian-MB, Russian-Phonetic-Max) werden entfernt
  • Quarantäne-Attribute werden gelöscht
  • Keyboard-Caches werden bereinigt
  • Das Layout wird sofort im System registriert (TIS)

Voraussetzungen

  • macOS 12 (Monterey) oder neuer
  • Administrator-Passwort (wird beim Installieren abgefragt)
